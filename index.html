Usage Guide:-

To use this plugin, add following link and script to head of your HTML document then add class name socialShareBtn to any Button element.

To add custom button or custom message, add window.customShare and window.customMessage to your file before loading scripts.

You are also free to apply any style or include anything in the Share button, it just need to have class name as socialShareBtn.
